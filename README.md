# Algorithm

一个用 Go 语言编写的算法学习项目，包含 LeetCode、洛谷、牛客等平台的算法题解以及常用数据结构的实现。

## 目录结构

```
algorithm/
├── dataStructure/       # 数据结构实现
├── LeetCode/           # LeetCode 题解
│   ├── array_and_hash/ # 数组和哈希表
│   ├── binary/         # 二进制相关
│   ├── string_practice/ # 字符串练习
│   ├── slide_window/   # 滑动窗口
│   ├── differential_array/ # 差分数组
│   ├── dynamic_program/    # 动态规划
│   └── backtrack_dfs_bfs/  # 回溯/DFS/BFS
├── doubao/             # 豆包题库
├── luogu/              # 洛谷题库
├── niuke/              # 牛客题库
└── go.mod              # Go 模块文件
```

## 数据结构

项目实现了以下基础数据结构：

| 数据结构 | 文件 |
|---------|------|
| 栈 | `dataStructure/Stack.go` |
| 双端队列 | `dataStructure/Deque.go` |
| 集合 | `dataStructure/Set.go` |
| 单链表 | `dataStructure/SingleList.go` |
| 链表节点 | `dataStructure/LinkedNode.go` |

## 学习路径

建议按照以下顺序学习：
1. **数据结构基础** - 先掌握栈、队列、链表等基本数据结构
2. **数组和哈希** - 理解基本的数据处理技巧
3. **双指针与滑动窗口** - 学习优化时间复杂度的方法
4. **二进制操作** - 掌握位运算技巧
5. **动态规划** - 解决复杂的最优化问题
6. **回溯与搜索** - 学习系统性的遍历方法
